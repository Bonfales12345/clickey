# Clickey

A lightweight, background-native menu bar utility that triggers audio on global keystrokes using CoreGraphics Event Taps.

</div>

---

### Requirements
*   **Permissions:** Must be granted **Accessibility** privileges in *System Settings > Privacy & Security > Accessibility*.

---

### Installation
1.  Download the latest release.
2.  Move `Clickey.app` to your `/Applications` folder.
3.  Launch the app and grant Accessibility permissions when prompted.
