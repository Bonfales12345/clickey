# Clickey

Clickey is a Open Source lightweight, background-native menu bar utility that play a mechanical keyboard switch sound on global keystrokes. Only for macOS (Apple Silicon).

### Requirements
*   **Device:** An Apple Silicon Mac (Intel Macs is **NOT** supported)
*   **Permissions:** Must be granted **Accessibility** privileges in *System Settings > Privacy & Security > Accessibility*.

---

### Installation
1.  Download the latest release.
2.  Extract the .zip File
3.  Move `Clickey.app` to your `/Applications` folder.
4.  Launch the app and grant Accessibility permissions when prompted.
