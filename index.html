Clickey

An open-source, lightweight, background-native menu bar utility that triggers a mechanical keyboard switch sound on global keystrokes.

View on GitHub

Satisfying Sounds

Triggers mechanical keyboard switch sound when you're typing by default. You can customize it to what sound you like. Just download the Source code and then change the audio file.

Lightweight App

Designed to run silently in the background with near-zero CPU and RAM utilization.

Fully Open Source

The app is free forever. Clone the repository, tweak the source, and build your ideal typing experience.
